# Issue #4: 用 Gemini 生成 UI 设计图

**Status:** OPEN
**Link:** https://github.com/Davie521/your-podcast/issues/4

## 目标

使用 Gemini 生成几个核心页面的简单设计图，作为前端开发的参考。

## 需要的设计图

### 1. 登录界面
- 登录/注册页面
- 简洁风格，符合播客 App 调性

### 2. 个人页
- 用户头像、昵称
- 收藏/历史播放记录
- 个人设置入口

### 3. Onboarding — 选择兴趣
- 新用户首次登录后的引导流程
- 选择感兴趣的科技话题标签（AI、前端、后端、产品、创业等）
- 根据兴趣推荐播客内容

### 4. 所有人界面（首页/发现页）
- 播客列表（每日推荐 + 往期）
- 搜索/筛选
- 公共内容展示，无需登录即可浏览

## 方式

- 使用 Gemini 生成设计概念图 / wireframe
- 输出存放到 `doc/designs/` 目录
